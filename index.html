What is CSS?

  1. CSS stands for Cascading Style Sheets.

  2. CSS describes how HTML element are to be displyed on screen,paper,or in other media.

  3. CSS is not neccesary CSS saves a lot of work. can control the layout of multiple web pages all at once

  4. External stylesheet are stored in css files.

CSS can be added to HTML documents in 3 ways:

  • Inline-by using the style attribute inside HTML elements
  • Internal-by using a style element in the head of section
  • External-by using a link element to link to an external CSS file.However, in this tutorial we will use inline and internal style, because this is easier to demonstrate,and easier for you to try it yourself.

When comparing CSS class vs ID,the difference is that CSS class applies a style to multiple element.ID,on the other hand,applies a style to one unique element.ID is also special in that you can use a special URL to link directly to an element and its used by javascript.

In CSS,selectors are used to target a specific element or range of element on a web page.once an element has been targeted,a style or set of style can be applied to the element.

There is a wide range of selectors available.Two of the most commonly used are class and ID.Both are used to target element to which a style should be applied.
